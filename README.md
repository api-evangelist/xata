# Xata (xata)

Xata is a serverless Postgres platform built for modern development and agentic workloads. It enables instant database branching with copy-on-write storage, scale-to-zero compute, built-in PII anonymization, and a REST management API with TypeScript and Python SDKs. Deployment options include fully managed SaaS (Xata Cloud), Bring Your Own Cloud (BYOC), and open-source self-hosted.
